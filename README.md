# swengineer

## Install

```
pnpm install
```

## Build

```
pnpm run build
```

### Development

Uses files under `client/src/`.

http://localhost:3000/

#### Debug

```
pnpm run dev
```

### Production

Uses files created for build under `client/build/`.

http://localhost:8080/

#### public/ directory

http://localhost:8080/public/

## Test

```
pnpm test
```
